Learn the Fundamentals

While there are hundreds of computer programming languages in use today (you might be familiar with names like Java, JavaScript, Python, PHP, Ruby, or C++), most computer programming languages share the same foundational building blocks.

We are here to help you learn just what these programming building blocks are, how they work, and how teaching them can help learners discover one of the most beneficial skill sets out there: coding.

Learn React

React offers graceful solutions to some of front-end programming’s most persistent issues, allowing you to build dynamic and interactive web apps with ease. It’s fast, scalable, flexible, powerful, and has a robust developer community that’s rapidly growing.

You’ll develop a strong understanding of React’s most essential concepts: JSX, class and function components, props, state, lifecycle methods, and hooks. You’ll be able to combine these ideas in React’s modular programming style.
